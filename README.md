# Angular-E-Cart

A beautiful website which show list of available footwear from the website.

### Learning from this application:
  - Angular Ecosystem
  - Component
  - Templates
  - One way data binding
    - string interpolation:{{}}
    - Event binding: (click)
   
  - Two way data binding: `ngModel`
  - `ngFor` `ngIf` `ngStyle` `ngClass`
  - `@Input` & `@Output`
